# GPM 派車系統文件

歡迎使用 **GPM 派車系統** 技術文件。本網站提供系統架構、RESTful API 說明、版本更新履歷與常見問題，協助您快速整合 AGV 派車功能。

## 典型整合流程

1. 向系統管理員取得 API 認證 Token
2. 查詢可用車輛與站點資訊
3. 建立派車任務並追蹤狀態
4. 任務完成後接收回報或 Webhook 通知

:::tip 開始整合
建議先閱讀 [API 概覽](/docs/api/overview)，再參考 [VMS 提供的 API](/docs/api/vms-agv) 或 [AGV 提供的 API](/docs/api/agv-api) 進行整合測試。
:::
