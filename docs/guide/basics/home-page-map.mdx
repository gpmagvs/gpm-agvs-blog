# 主頁地圖


## 功能概述

主頁地圖以二維平面圖顯示廠區路徑、站點與 AGV 即時位置，是監控派車、確認路徑佔用與手動派工的主要視覺介面。

## 進入方式

1. 進入 [主畫面](/docs/guide/basics/home-page)
2. 中央區域即為主頁地圖（登入後預設顯示）

## 畫面說明

![主頁地圖](/img/webui/home-page-map.png)

### 地圖元素

| 元素 | 說明 |
|------|------|
| 節點（Node） | 圓形編號點，代表路徑交會或定位 Tag（如 9、21、47、55） |
| 路徑（Path） | 連線節點的線段，為 AGV 可行駛路線 |
| AGV 圖示 | 顯示車輛編號與狀態；離線為灰色、閒置／執行等以不同顏色區分 |
| 站點 | 矩形標籤，如 `AOI-016`～`AOI-021`、`STK#1_L/R`、`QV#7_L/R` 等 |
| 區域 | 地圖上的分割槽標示，如 `Z1`、`Z2-1`、`Z2-2`、`Z3` |
| 保留標示 | 站點被任務佔用時可能顯示 **(Reserved)** 等提示 |

### 地圖工具列

地圖左上角浮動按鈕：

| 按鈕 | 說明 |
|------|------|
| ＋ | 放大地圖 |
| － | 縮小地圖 |
| 首頁圖示 | 重置視角至預設位置 |
| 齒輪 | 地圖相關設定 |

地圖右上角顯示目前遊標或視角座標（如 `(-6.82, -7.07)`）及 **Grid Size**（格距，如 `10m`）。

### 右側顯示設定

| 專案 | 說明 |
|------|------|
| 顯示模式 | Index、Name、Name & Tag、Tag 等站點標籤顯示方式 |
| 地圖模式 | SLAM 地圖切換 |
| 顯示 AGV | 是否顯示 AGV 圖示 |
| AGV 電量 | 是否在地圖上顯示電量 |
| 網格顯示 | 是否顯示座標網格 |
| 背景／區域／圖例 | 控制背景、分割槽與圖例顯示 |
| Rack Display | 是否顯示料架 |
| Pan/Zoom | 是否允許拖曳與縮放地圖 |

### 任務派送

地圖右下 **任務派送** 按鈕可開啟手動派工對話方塊，指定動作、車輛、來源與目的地後送出任務。

## 操作步驟

### 縮放與移動地圖

1. 使用左上角 **＋／－** 調整比例
2. 確認 **Pan/Zoom** 為 Enable 時，可直接拖曳地圖
3. 點選 **首頁圖示** 恢復預設視角

### 檢視 AGV 位置

1. 確認右側 **顯示 AGV** 已開啟
2. 在地圖上找到 AGV 圖示（如 `AGV_002`）
3. 對照 [車輛狀態](/docs/guide/basics/vehicle-status) 面板中的連線與閒置／執行狀態

### 手動派送任務

1. 點選地圖右下角 **任務派送**
2. 選擇 **動作**（如搬運）、**車輛**、**來源**、**目的地**
3. 確認引數後派送；任務將出現在 [任務列表](/docs/guide/basics/task-list) 的未完成分頁

## 注意事項

- 地圖上 AGV 狀態與左側 **車輛狀態** 應一致；若不同步請確認 VMS 連線
- 站點標示為黃色或帶 **(Reserved)** 時，表示可能被任務佔用，派工前請確認
- 關閉 **顯示 AGV** 或 **AGV 電量** 僅影響地圖顯示，不影響實際監控資料
- 手動 **任務派送** 前請確認派工模式與 HOST 連線狀態是否符合現場規範
