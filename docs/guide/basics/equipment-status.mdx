# 裝置狀態


## 功能概述

裝置狀態面板以表格列出產線裝置（如 AOI、STK、QV 等）的即時狀態，包含運轉狀態、移載需求、帳籍（有無貨物）等資訊，供現場人員掌握各站點是否可派車收送料。

## 進入方式

1. 進入 [主畫面](/docs/guide/basics/home-page)
2. 確認左側面板已展開
3. 點選 **裝置狀態** 分頁

## 畫面說明

![裝置狀態面板](/img/webui/equipment-status.png)

### 工具列

| 專案 | 說明 |
|------|------|
| 全部展開 | 展開所有裝置列的詳細資訊 |
| 搜尋 | 依裝置名稱或 Tag 快速篩選 |
| 顯示模式 | 切換裝置狀態的顯示方式（預設為 **裝置狀態**） |

### 表格欄位

| 欄位 | 說明 |
|------|------|
| 裝置名稱 | 裝置／站點 ID（如 `AOI-016`、`QV#7_L`），左側圖示代表裝置型別 |
| Tag | 裝置對應之地圖 Tag 編號 |
| 主狀態 | 裝置運轉狀態，正常時顯示 **NORMAL**（綠色） |
| 移載需求 | 裝置對 AGV 的收送料請求狀態 |
| 帳籍狀態 | 站點上是否有貨物 |
| 貨物 ID | 目前站點載具／貨物識別碼；可點選圖示檢視或編輯 |
| 貨物種類 | 貨物型別；無資料時顯示 `---` |

### 移載需求狀態

| 狀態 | 說明 | 標示 |
|------|------|------|
| No-Request | 目前無移載請求 | 灰色 |
| 可收料 | 裝置請求 AGV 送料至該站 | 綠色 |

### 帳籍狀態

| 狀態 | 說明 | 標示 |
|------|------|------|
| 無貨 | 站點上無貨物 | 橘色 |
| 有貨 | 站點上有貨物 | 綠色 |

## 操作步驟

### 查詢裝置狀態

1. 進入 **裝置狀態** 分頁
2. 在搜尋框輸入裝置名稱或 Tag 關鍵字
3. 確認 **主狀態**、**移載需求**、**帳籍狀態** 是否符合預期

### 檢視貨物資訊

1. 找到目標裝置列
2. 若 **帳籍狀態** 為 **有貨**，檢視 **貨物 ID** 欄
3. 點選貨物 ID 旁圖示，開啟詳情或編輯視窗（依許可權）

### 展開全部裝置

1. 點選 **全部展開**
2. 瀏覽各裝置完整狀態
3. 配合中央地圖對照 Tag 位置與 AGV 路徑

## 注意事項

- **主狀態** 非 NORMAL 時，該裝置可能無法正常派工，請先排除裝置異常
- **可收料** 表示裝置已發出移載請求，排程系統將依規則指派 AGV；請勿重複手動派工造成衝突
- **帳籍狀態** 與實際現場不一致時，請依 SOP 更新帳籍或聯絡工程人員
- 裝置狀態與 [任務列表](/docs/guide/basics/task-list) 連動：派工前建議同時確認目標站點的移載需求與帳籍
- 若頂部 **CIM** 或 **SECS Platform** 指示異常，裝置狀態可能無法與 HOST 同步
