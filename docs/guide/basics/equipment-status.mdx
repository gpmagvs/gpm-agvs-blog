# 設備狀態

對應前端模組：`Index` 左側面板 · **設備狀態** 分頁

## 功能概述

設備狀態面板以表格列出產線設備（如 AOI、STK、QV 等）的即時狀態，包含運轉狀態、移載需求、帳籍（有無貨物）等資訊，供現場人員掌握各站點是否可派車收送料。

## 進入方式

1. 進入 [主畫面](/docs/guide/basics/home-page)
2. 確認左側面板已展開
3. 點選 **設備狀態** 分頁

## 畫面說明

![設備狀態面板](/img/webui/equipment-status.png)

### 工具列

| 項目 | 說明 |
|------|------|
| 全部展開 | 展開所有設備列的詳細資訊 |
| 搜尋 | 依設備名稱或 Tag 快速篩選 |
| 顯示模式 | 切換設備狀態的顯示方式（預設為 **設備狀態**） |

### 表格欄位

| 欄位 | 說明 |
|------|------|
| 設備名稱 | 設備／站點 ID（如 `AOI-016`、`QV#7_L`），左側圖示代表設備類型 |
| Tag | 設備對應之地圖 Tag 編號 |
| 主狀態 | 設備運轉狀態，正常時顯示 **NORMAL**（綠色） |
| 移載需求 | 設備對 AGV 的收送料請求狀態 |
| 帳籍狀態 | 站點上是否有貨物 |
| 貨物 ID | 目前站點載具／貨物識別碼；可點選圖示查看或編輯 |
| 貨物種類 | 貨物類型；無資料時顯示 `---` |

### 移載需求狀態

| 狀態 | 說明 | 標示 |
|------|------|------|
| No-Request | 目前無移載請求 | 灰色 |
| 可收料 | 設備請求 AGV 送料至該站 | 綠色 |

### 帳籍狀態

| 狀態 | 說明 | 標示 |
|------|------|------|
| 無貨 | 站點上無貨物 | 橘色 |
| 有貨 | 站點上有貨物 | 綠色 |

## 操作步驟

### 查詢設備狀態

1. 進入 **設備狀態** 分頁
2. 在搜尋框輸入設備名稱或 Tag 關鍵字
3. 確認 **主狀態**、**移載需求**、**帳籍狀態** 是否符合預期

### 查看貨物資訊

1. 找到目標設備列
2. 若 **帳籍狀態** 為 **有貨**，查看 **貨物 ID** 欄
3. 點選貨物 ID 旁圖示，開啟詳情或編輯視窗（依權限）

### 展開全部設備

1. 點選 **全部展開**
2. 瀏覽各設備完整狀態
3. 配合中央地圖對照 Tag 位置與 AGV 路徑

## 注意事項

- **主狀態** 非 NORMAL 時，該設備可能無法正常派工，請先排除設備異常
- **可收料** 表示設備已發出移載請求，調度系統將依規則指派 AGV；請勿重複手動派工造成衝突
- **帳籍狀態** 與實際現場不一致時，請依 SOP 更新帳籍或聯繫工程人員
- 設備狀態與 [任務列表](/docs/guide/basics/task-list) 連動：派工前建議同時確認目標站點的移載需求與帳籍
- 若頂部 **CIM** 或 **SECS Platform** 指示異常，設備狀態可能無法與 HOST 同步
