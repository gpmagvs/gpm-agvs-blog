# 登入


## 功能概述

登入頁為 AGVS Web 介面的入口，使用者需輸入帳號與密碼通過驗證後，才能進入主畫面進行派車監控、帳籍管理與系統設定。不同帳號可能具備不同操作許可權。

## 進入方式

在瀏覽器開啟 AGVS 網址：

```
http://{AGVS_HOST}/
```

實際網址與 Port 依現場部署。若尚未登入，系統會自動顯示登入頁。

## 畫面說明

![登入頁](/img/webui/login-page.png)

登入頁分為左側 **登入表單** 與右側 **系統特色** 兩區。

### 左側登入表單

| 區域 | 說明 |
|------|------|
| GPM Logo | 系統品牌識別 |
| AGV Control System | 系統名稱副標題 |
| 使用者名稱 | 輸入帳號（Username） |
| 密碼 | 輸入密碼（Password） |
| 登入 | 送出登入請求 |
| Back to Home | 返回首頁（Landing Page） |

### 右側特色說明

右側面板列出 AGVS 主要功能摘要：

| 專案 | 說明 |
|------|------|
| Automated Path Planning | 自動路徑規劃 |
| Real-Time Monitoring and Management | 即時監控與管理 |
| Multi-Vehicle Coordination and Scheduling | 多車協調與排程 |

## 操作步驟

### 登入系統

1. 在瀏覽器開啟 AGVS 網址
2. 於 **使用者名稱** 欄位輸入帳號
3. 於 **密碼** 欄位輸入密碼
4. 點選 **登入**
5. 驗證成功後，系統導向 [主畫面](/docs/guide/basics/home-page)

### 返回首頁

若需回到 Landing Page（功能入口頁），點選 **Back to Home**。

## 注意事項

- 帳號與密碼請向系統管理員取得；首次使用建議確認帳號許可權範圍
- 登入失敗時請確認帳號、密碼是否正確，以及 Caps Lock 是否開啟
- 長時間未操作可能 Session 過期，需重新登入
- 請勿在公用電腦勾選記住密碼或留存登入狀態
- 登入後可從右上角使用者資訊確認目前登入帳號與角色
