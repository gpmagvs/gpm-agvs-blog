# 主畫面

對應前端模組：`Index`（主監控畫面）

## 功能概述

主畫面為 AGVS 派車系統的核心操作介面，提供地圖監控、車輛狀態、設備資訊與派工相關功能入口。

## 進入方式

登入成功後自動進入主畫面；或從左側選單返回主畫面。

## 畫面說明

![AGVS 主畫面](/img/webui/home-page.png)

| 區域 | 說明 |
|------|------|
| 頂部狀態列 | 服務狀態（VMS、CIM、SECS Platform）、[連線與模式切換](/docs/guide/basics/modes-change) |
| 左側面板 | [車輛狀態](/docs/guide/basics/vehicle-status)、[任務列表](/docs/guide/basics/task-list)、[設備狀態](/docs/guide/basics/equipment-status) |
| 中央地圖 | [主頁地圖](/docs/guide/basics/home-page-map)：AGV 位置、站點、路徑與地圖操作工具 |
| 右側面板 | 地圖顯示設定、圖例與任務派送 |
| 底部狀態列 | 儲存空間、快速操作、版本資訊與系統時間 |
