# 車輛狀態

對應前端模組：`Index` 左側面板 · **車輛狀態** 分頁

## 功能概述

車輛狀態面板顯示所有 AGV 的即時資訊，包含連線狀態、運行狀態、所在位置、電量與任務概況，供現場人員快速掌握車隊狀況。

## 進入方式

1. 進入 [主畫面](/docs/guide/basics/home-page)
2. 確認左側面板已展開
3. 點選 **車輛狀態** 分頁（預設分頁）

:::info 同面板其他分頁
左側面板另含 [任務列表](/docs/guide/basics/task-list)、[設備狀態](/docs/guide/basics/equipment-status) 分頁。
:::

## 畫面說明

![車輛狀態面板](/img/webui/vehicle-status.png)

每輛 AGV 以卡片方式列出，主要欄位如下：

| 欄位 | 說明 |
|------|------|
| 車輛名稱 | AGV 編號（如 `AGV_001`） |
| 連線狀態 | **已上線**（Online）／**離線**（Offline），以指示燈顏色區分 |
| 運行狀態 | 初始化、閒置、運行、當機、充電等（見下表） |
| 位置 | 目前所在站點或 Tag（如 `Charge_AOI_1`） |
| 電量 | 電池百分比；充電中時會顯示充電狀態 |
| Simulation | 切換該車是否為模擬模式 |
| 定位 | 對 AGV 執行定位操作 |

### 運行狀態對照

| 狀態 | 說明 | 指示 |
|------|------|------|
| 初始化 | 車輛啟動或重連中 | 黃色 |
| 閒置 | 已上線、待命中 | 黃色 |
| 運行 | 執行任務或移動中 | 綠色 |
| 當機 | 異常或停機 | 紅色 |
| 充電 | 正在充電 | 藍色 |

## 操作步驟

### 查看單一 AGV 詳情

1. 在車輛狀態列表中找到目標 AGV
2. 確認連線狀態、運行狀態、位置與電量
3. 若需在地圖上定位，可點選該車卡片或地圖上對應 AGV 圖示

### 切換 Simulation 模式

1. 找到目標 AGV 卡片上的 **Simulation** 開關
2. 開啟後該車以模擬模式運作（測試／除錯用途）
3. 正式派工前請確認已關閉 Simulation

### 執行定位

1. 點選 AGV 卡片上的 **定位** 按鈕
2. 系統向 VMS 送出定位請求
3. 等待 AGV 回報定位結果，並確認地圖上位置是否正確

## 注意事項

- 若頂部 **VMS** 指示為紅色，車輛狀態可能無法即時更新，請先排除 VMS 連線問題
- 顯示 **VMS 斷線** 時，請聯繫工程人員確認 VMS 與 AGV 通訊
- 電量過低時系統可能自動派發充電任務，請勿在充電過程中強制中斷（除非 SOP 允許）
- Simulation 模式下的 AGV 不應參與正式生產派工
