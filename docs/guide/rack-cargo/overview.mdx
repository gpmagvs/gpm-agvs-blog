# 帳籍管理頁面

對應前端模組：`RacksManager`（左側選單 **帳籍管理**）

## 功能概述

帳籍管理頁面用於檢視與維護各 Zone 內 RACK 儲位的貨物帳籍，包含貨物 ID、在架狀態、空／滿櫃、放站時間與貨物類型等資訊，並提供 Zone 水位、出貨順序與 RACK 參數等管理功能。

## 進入方式

1. 登入 AGVS Web 介面
2. 點選左側選單 **帳籍管理**

## 畫面說明

![帳籍管理頁面](/img/webui/rack-cargo-managment.png)

### 頂部工具列

| 項目 | 說明 |
|------|------|
| Zone 分頁 | 切換不同系統／區域，如 `SYS0029 (Zone1)`、`SYS0030 (Zone2)`、`SYS0031 (Zone3)` |
| 清除所有無料帳籍 | 批次清除無貨物資料的帳籍紀錄 |
| 低水位提報設置 | 設定低水位告警或提報規則 |
| RACK 管理參數設定 | 調整 RACK 管理相關系統參數 |
| 顯示模式 | 切換儲位卡片或列表的顯示方式 |
| 分頁顯示 | 控制每頁顯示的儲位數量 |

### Zone 概覽

| 項目 | 說明 |
|------|------|
| 水位 | 目前占用／總容量，如 **0/2** 表示 2 個儲位中 0 個有貨 |
| 設置下料終點設備選項 | 設定該 Zone 下料終點設備 |
| 變更出貨順序 | 調整出貨或派工優先順序 |

### 儲位卡片（Slot）

每個編號儲位（如 **01**、**02**）以卡片呈現：

| 欄位 | 說明 |
|------|------|
| 啟用 | 藍色開關，控制該儲位是否啟用 |
| 貨物 ID | 目前帳籍上的貨物／載具識別碼；無貨時為空 |
| 貨物在架 | **有貨**／**無貨**（無貨時常以紅色標示） |
| 空／滿櫃 | **空櫃** 或 **滿櫃** 狀態 |
| 放站時間 | 貨物放置或更新至該儲位的時間 |
| 貨物類型 | 貨物種類；未設定時顯示 **未設定貨物** |
| 齒輪圖示 | 開啟該儲位的個別設定 |

## 操作步驟

### 切換 Zone

1. 點選頂部 Zone 分頁（如 **SYS0030 (Zone2)**）
2. 確認 **水位** 與下方儲位卡片已更新為該 Zone 資料

### 查看儲位帳籍

1. 在 Zone 概覽區找到目標儲位編號（如 **01**）
2. 確認 **貨物在架**、**空／滿櫃**、**貨物 ID** 是否與現場一致
3. 若需修改單一儲位，點選卡片左下角 **齒輪** 進入設定

### 啟用／停用儲位

1. 找到目標儲位卡片
2. 切換 **啟用** 開關
3. 停用後該儲位通常不參與派工或帳籍更新，請依 SOP 操作

## 注意事項

- 帳籍資料與 [設備狀態](/docs/guide/basics/equipment-status) 的 **帳籍狀態**、**貨物 ID** 應保持一致；不一致時請先確認 MCS／HOST 同步
- **清除所有無料帳籍** 為批次操作，執行前請確認不會影響進行中任務
- 修改 **RACK 管理參數設定** 或 **出貨順序** 可能影響自動派工，建議由授權人員操作
- 頂部若有 **設備警報** 列，請先評估是否允許變更帳籍後再操作
