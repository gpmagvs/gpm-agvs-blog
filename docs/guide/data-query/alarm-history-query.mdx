# 警報歷史查詢


## 功能概述

警報歷史查詢用於依時間區間與多種條件，查詢 AGV、充電站、VMS 等裝置的歷史警報紀錄，包含異常碼、警報描述、關聯任務與排除方法等，並支援關鍵字搜尋與 CSV 匯出。

## 進入方式

1. 登入 AGVS Web 介面
2. 點選左側選單 **資料查詢**
3. 進入 **警報歷史查詢** 頁面

## 畫面說明

![警報歷史查詢](/img/webui/alarm-history-query.png)

### 查詢條件

| 欄位 | 說明 |
|------|------|
| 起始時間 | 查詢區間開始時間 |
| 結束時間 | 查詢區間結束時間 |
| 異常型別 | 警報型別篩選；`ALL` 表示不篩選 |
| AGV 名稱 | 指定 AGV；`ALL` 表示不篩選 |
| 任務名稱 | 關聯任務篩選；`ALL` 表示不篩選 |
| 異常描述 | 依描述關鍵字篩選 |
| 搜尋 | 依上述條件查詢 |
| 關鍵字 | 額外關鍵字搜尋欄位 |
| 關鍵字搜尋 | 以關鍵字執行搜尋 |
| 輸出 csv 檔 | 將查詢結果匯出為 CSV |

### 結果表格

| 欄位 | 說明 |
|------|------|
| 發生時間 | 警報觸發時間 |
| 裝置名稱 | 警報來源，如 `AGV_002`、`Charge_AOI_1`、`VMS` |
| 異常碼 | 系統定義的警報程式碼（如 1075、7010、1003） |
| 警報描述 | 中英文警報說明（如充電站 IO 模組連線異常） |
| 異常型別 | 通常顯示 **Alarm**（紅色標籤） |
| 任務名稱 | 若與任務相關，顯示任務 ID；旁有複製圖示 |
| 發生地點 | 警報發生位置 Tag 或編號 |
| 排除方法 | 建議處置方式（如「請洽 GPM」） |
| 持續時間(s) | 警報持續秒數 |
| 清除警報人員 | 清除該警報的操作人員 |

:::info 表格列標示
警報紀錄列常以淺紅色背景標示，方便與一般資料區分。
:::

## 操作步驟

### 查詢警報紀錄

1. 設定 **起始時間** 與 **結束時間**
2. 依需要選擇 **異常型別**、**AGV 名稱**、**任務名稱** 或填寫 **異常描述**
3. 點選 **搜尋**
4. 在表格中檢視結果；使用底部分頁瀏覽

### 關鍵字搜尋

1. 在 **關鍵字** 欄輸入欲查詢的文字（如異常碼、裝置名稱片段）
2. 點選 **關鍵字搜尋**
3. 檢視篩選後的警報列表

### 複製關聯任務 ID

1. 在 **任務名稱** 欄找到目標任務
2. 點選旁邊複製圖示
3. 可至 [任務歷史查詢](/docs/guide/data-query/task-history-query) 進一步查詢該任務詳情

### 匯出 CSV

1. 完成查詢並確認結果
2. 點選 **輸出 csv 檔**
3. 儲存檔案供離線分析或報表使用

## 注意事項

- 本頁為 **歷史** 警報；即時告警請至主畫面頂部告警列或 [告警處理](/docs/guide/maintenance/alarm-handling) 相關功能處置
- **排除方法** 若顯示「請洽 GPM」，表示需工程支援，請記錄 **異常碼** 與 **發生時間** 後聯絡
- 同一異常可能多次出現（如充電站 IO 連線異常），可透過 **裝置名稱** 與 **異常碼** 交叉篩選分析
- 查詢區間過大可能增載入入時間，建議依追溯需求設定合理範圍
- 匯出資料可能含現場資訊，請依公司資安規範儲存
