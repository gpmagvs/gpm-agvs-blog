# 交管熱點查詢


## 功能概述

交管熱點查詢以地圖熱力圖（Heatmap）視覺化 AGV 行駛或交管佔用密集區域，協助分析瓶頸路段、壅塞節點與路徑規劃問題。

## 進入方式

1. 登入 AGVS Web 介面
2. 點選左側選單 **資料查詢**
3. 進入 **交管熱點查詢** 頁面

## 畫面說明

![交管熱點查詢](/img/webui/traffic-status-query.png)

### 地圖區

| 元素 | 說明 |
|------|------|
| 節點與路徑 | 與主畫面相同的路網結構（編號節點、站點標籤） |
| 熱力圖 | 以綠→黃→橘→紅表示流量或佔用強度，紅色為熱點核心 |
| 座標／格距 | 右上角顯示視角座標與 Grid Size（如 `10m`） |
| 縮放工具 | 左上角 **＋／－** 與重置視角 |

### 右側顯示設定

| 專案 | 說明 |
|------|------|
| 圖例顯示 | 是否顯示地圖圖例 |
| Rack 顯示 | 是否顯示料架 |
| Pan/Zoom | 啟用地圖拖曳與縮放 |
| 搜尋 | 精確搜尋地圖點位 |

### 熱力圖調整（右下）

| 滑桿 | 說明 |
|------|------|
| Opacity | 熱力圖透明度 |
| Blur | 熱力邊緣模糊程度 |
| Radius | 每個資料點的影響半徑 |

## 操作步驟

### 檢視交管熱點

1. 進入頁面後等待地圖與熱力圖載入
2. 觀察紅色／橘色集中區域，即為高流量或高佔用節點（如節點 45、47、51 一帶）
3. 對照站點標籤（STK、AOI、充電站等）判斷是否為派工瓶頸

### 調整熱力圖顯示

1. 拖曳 **Opacity** 調整熱力圖與底圖的疊加效果
2. 調整 **Blur**、**Radius** 使熱點範圍更易辨識
3. 必要時開啟 **Rack 顯示** 或 **圖例顯示** 輔助對照

### 分析與改善

1. 記錄熱點所在 Tag 或站點
2. 搭配 [任務軌跡查詢](/docs/guide/data-query/traj-history-query) 確認是否為常走路徑交會
3. 若熱點對應交管佔用，可與工程人員檢討路徑或排程規則

## 注意事項

- 熱力圖反映查詢條件下之統計結果，非即時交管狀態；即時監控請用 [主頁地圖](/docs/guide/basics/home-page-map)
- 調整滑桿僅影響顯示，不改變原始統計資料
- 熱點集中在充電站或 STK 入口為常見現象，需結合派工量評估是否異常
- 分析報告建議截圖儲存，並註明查詢時間區間（若頁面有提供）
