# 任務軌跡查詢

對應前端模組：軌跡查詢（左側選單 **資料查詢** · 任務軌跡）

## 功能概述

任務軌跡查詢用於查詢已完成或已取消任務的 AGV 行駛路徑，左側以條件篩選任務列表，右側地圖以軌跡線顯示實際走過路線，並支援軌跡回放、顏色／線寬調整與儲存軌跡圖。

## 進入方式

1. 登入 AGVS Web 介面
2. 點選左側選單 **資料查詢**
3. 進入 **任務軌跡查詢** 頁面

## 畫面說明

![任務軌跡查詢](/img/webui/traj-history-query.png)

頁面分為 **左側查詢／任務列表** 與 **右側軌跡地圖** 兩區。

### 頂部分頁

| 分頁 | 說明 |
|------|------|
| 單次任務軌跡查詢 | 依條件查詢並顯示單一或多筆任務軌跡 |
| 軌跡回顧 | 軌跡回顧相關功能（依現場設定） |

### 左側查詢條件

| 欄位 | 說明 |
|------|------|
| 車輛選擇 | 下拉選擇 AGV（如 `AGV_001`） |
| 起始時間 | 查詢區間開始時間 |
| 結束時間 | 查詢區間結束時間 |
| 任務 ID | 指定任務編號（可選） |
| 查詢 | 執行搜尋並更新下方任務表格 |

### 任務列表

| 欄位 | 說明 |
|------|------|
| No | 序號 |
| 任務 ID | 任務唯一識別碼 |
| 任務狀態 | **完成**（綠）、**取消**（灰）等 |
| 來源／目的地 | 任務起訖站點 |
| 車輛 | 執行 AGV |
| 開始／結束時間 | 任務起訖時間 |
| 花費時間 | 任務耗時（秒） |
| 顯示軌跡 | 點選眼睛圖示或選取列，於右側地圖顯示該任務軌跡 |

### 右側軌跡地圖

| 項目 | 說明 |
|------|------|
| 縮放／首頁 | 地圖左上角 **＋／－** 與重置視角 |
| 軌跡顏色 | 調整軌跡線顏色（預設藍色） |
| 軌跡寬度 | 調整軌跡線粗細（如 12） |
| 儲存軌跡圖 | 將目前軌跡地圖儲存為圖檔 |
| 回放控制 | 倒帶、播放／暫停、快轉，重播 AGV 行駛過程 |
| 軌跡線 | 藍色（或自訂顏色）路徑，表示該任務實際走過路線 |
| 標記點 | 地圖上紅色 pin 等標示特定位置 |
| 圖例標示 | 是否顯示地圖圖例 |
| Rack 顯示 | 是否顯示料架 |
| Pan/Zoom | 啟用地圖拖曳與縮放 |
| 搜尋 | 地圖上搜尋特定點位 |

## 操作步驟

### 查詢並顯示任務軌跡

1. 選擇 **車輛** 與 **起始／結束時間**
2. 可選填 **任務 ID** 縮小範圍
3. 點選 **查詢**
4. 在任務列表中點選 **顯示軌跡**（眼睛圖示）
5. 右側地圖顯示該任務的藍色軌跡線

### 回放軌跡

1. 完成軌跡顯示後，使用地圖上方 **回放控制** 按鈕
2. 播放 AGV 沿軌跡移動的過程
3. 必要時暫停或調整播放速度（快轉／倒帶）

### 調整軌跡顯示

1. 變更 **軌跡顏色** 或 **軌跡寬度** 以便辨識
2. 開關 **Rack 顯示**、**圖例標示** 輔助對照站點
3. 使用 **儲存軌跡圖** 保存目前畫面

### 交叉查詢任務詳情

1. 記下任務列表中的 **任務 ID**
2. 至 [任務歷史查詢](/docs/guide/data-query/task-history-query) 查詢該任務完整紀錄
3. 若任務期間有警報，可至 [警報歷史查詢](/docs/guide/data-query/alarm-history-query) 對照時間

## 注意事項

- 僅 **已完成** 或 **已取消** 等已結束任務通常才有完整軌跡；進行中任務請至 [主頁地圖](/docs/guide/basics/home-page-map) 即時監控
- 查詢區間過大可能返回大量任務，建議配合 **任務 ID** 或縮短時間範圍
- 軌跡為歷史紀錄，與現場即時位置可能不同
- **儲存軌跡圖** 供分析或報告使用，請依公司規範保存
- 回放功能依瀏覽器效能，軌跡較長時可能需稍候載入
