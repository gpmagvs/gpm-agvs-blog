# AGV 電池歷史數據查詢

對應前端模組：電池歷史查詢（左側選單 **資料查詢** · AGV 電池歷史）

## 功能概述

AGV 電池歷史數據查詢以圖表呈現指定 AGV 在時間區間內的雙電池（電池 1／電池 2）歷史資料，包含電量、電壓、充電電流與放電電流，供維護與電池健康分析使用。

## 進入方式

1. 登入 AGVS Web 介面
2. 點選左側選單 **資料查詢**
3. 進入 **AGV 電池歷史數據查詢** 頁面

## 畫面說明

![AGV 電池歷史數據查詢](/img/webui/agv-bat-status-history-query.png)

### 查詢條件

| 欄位 | 說明 |
|------|------|
| AGV 名稱 | 下拉選擇車輛（如 `AGV_001`） |
| 開始時間 | 查詢區間起始日期時間 |
| 結束時間 | 查詢區間結束日期時間 |
| 查詢 | 載入並繪製圖表 |
| 圖表排列 | 切換網格／列表等版面配置 |

### 圖表區（2×2）

每張圖以 **電池 1**（橘線）與 **電池 2**（綠線）雙線顯示：

| 圖表 | Y 軸 | 說明 |
|------|------|------|
| 電量 | % | State of Charge，電量變化趨勢 |
| 電壓 | V | 電池電壓 |
| 充電電流 | A | 充電時電流；0 表示未充電 |
| 放電電流 | A | 放電時電流 |

各圖右上角提供縮放、平移、還原、選單等圖表工具。

## 操作步驟

### 查詢電池歷史

1. 選擇 **AGV 名稱**
2. 設定 **開始時間** 與 **結束時間**
3. 點選 **查詢**
4. 分別檢視四張圖表中的雙電池曲線

### 分析充放電行為

1. 在 **充電電流** 圖確認充電時段（電流大於 0）
2. 在 **放電電流** 圖確認執行任務時的放電量
3. 對照 **電量** 圖，確認充放電與 SOC 變化是否一致

### 比對異常

1. 若 **電壓** 兩電池差異過大，記錄時間點
2. 至 [警報歷史查詢](/docs/guide/data-query/alarm-history-query) 篩選同期電池或充電相關警報
3. 必要時對照 [車輛狀態](/docs/guide/basics/vehicle-status) 當下連線與充電狀態

## 注意事項

- 查詢區間過長可能使圖表資料點密集，可縮短區間或使用圖表縮放工具
- 充電電流為 0 且電量下降，表示該時段僅放電未充電，屬正常現象
- 雙電池 AGV 應同時比對兩條曲線；單側異常可能需更換或校正電池模組
- 歷史資料供趨勢分析，即時電量請以 [車輛狀態](/docs/guide/basics/vehicle-status) 或 [主頁地圖](/docs/guide/basics/home-page-map) 為準
- 圖表匯出或截圖請依公司資安規範保存
