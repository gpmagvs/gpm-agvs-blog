# 任務歷史查詢

對應前端模組：任務查詢（左側選單 **資料查詢** · 任務歷史）

## 功能概述

任務歷史查詢用於依時間區間與多種條件，检索已結束的 AGV 任務紀錄，包含執行結果、起訖站點、貨物 ID 與失敗原因等，並支援匯出 CSV 供離線分析。

## 進入方式

1. 登入 AGVS Web 介面
2. 點選左側選單 **資料查詢**
3. 進入 **任務歷史查詢** 頁面

## 畫面說明

![任務歷史查詢](/img/webui/task-history-query.png)

### 查詢條件

| 欄位 | 說明 |
|------|------|
| 起始時間 | 查詢區間開始時間 |
| 結束時間 | 查詢區間結束時間 |
| 任務 ID | 指定任務編號；`ALL` 表示不篩選 |
| Carrier ID | 載具／貨物 ID；`ALL` 表示不篩選 |
| 起終點搜尋 | 依起點或終點站點關鍵字篩選 |
| AGV 名稱 | 下拉選擇執行車輛 |
| 動作 | 任務類型（如搬運、充電）；`ALL` 表示不篩選 |
| 執行結果 | 完成、失敗、取消等；`ALL` 表示不篩選 |
| 異常描述 | 依失敗或異常說明關鍵字篩選 |
| 搜尋 | 依條件查詢並更新下方表格 |
| 輸出 csv 檔 | 將查詢結果匯出為 CSV |

### 結果表格

| 欄位 | 說明 |
|------|------|
| 任務 ID | 任務唯一識別碼 |
| 接收時間 | 任務建立時間 |
| 結束時間 | 任務結束時間 |
| 執行結果 | **完成**（綠）、**失敗**（紅）、**取消**（橘） |
| AGV 名稱 | 執行該任務的車輛 |
| 動作 | 任務類型，如 **搬運**、**充電** |
| 起點／起點 Slot | 任務來源站點與 Slot |
| 終點／終點 Slot | 任務目的站點與 Slot |
| 貨物 ID | 搬運任務的 Carrier ID |
| 取料時間 | 實際取貨時間 |
| 失敗原因 | 失敗或取消的詳細說明（如「用戶取消任務」） |

### 分頁與統計

表格下方顯示分頁控制與統計摘要，例如：

**共 N 筆，成功：X，失敗：Y，取消：Z**

## 操作步驟

### 查詢任務紀錄

1. 設定 **起始時間** 與 **結束時間**
2. 依需要填寫任務 ID、Carrier ID、AGV 名稱、動作、執行結果等條件
3. 點選 **搜尋**
4. 在表格中檢視符合條件的任務；使用底部分頁切換頁面

### 匯出 CSV

1. 完成查詢並確認結果正確
2. 點選 **輸出 csv 檔**
3. 儲存檔案至本機，供報表或追溯使用

### 分析失敗任務

1. 將 **執行結果** 設為 **失敗** 或 **取消**
2. 點選 **搜尋**
3. 查看 **失敗原因** 欄位，必要時搭配 **異常描述** 關鍵字縮小範圍

## 注意事項

- 查詢區間過大可能增加載入時間，建議依需求縮小時間範圍
- 本頁為 **歷史** 紀錄；進行中任務請至 [任務列表](/docs/guide/basics/task-list) 查看
- **輸出 csv 檔** 內容以目前查詢條件為準，變更條件後需重新搜尋再匯出
- 失敗原因若為「用戶取消任務」，表示人為中止，與設備異常需區分處理
- 匯出資料可能含生產資訊，請依公司資安規範保存與傳遞
