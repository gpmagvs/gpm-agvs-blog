# 稼動率查詢


## 功能概述

稼動率查詢用於統計 AGV 在指定日期區間內的狀態時間分佈，以圓餅圖呈現整體 RUN／IDLE／DOWN／CHARGE 等佔比，並以長條圖顯示各日明細，另提供 MTBF／MTTR 分析圖供可靠性評估。

## 進入方式

1. 登入 AGVS Web 介面
2. 點選左側選單 **資料查詢**
3. 進入 **稼動率查詢** 頁面

## 畫面說明

![稼動率查詢](/img/webui/avalitibility-query.png)

### 查詢條件（左側）

| 欄位 | 說明 |
|------|------|
| AGV 名稱 | 下拉選擇要統計的車輛（如 `AGV_001`） |
| 起始日期 | 統計區間開始日 |
| 結束日期 | 統計區間結束日 |
| 查詢 | 依條件重新載入圖表 |

### 分頁

| 分頁 | 說明 |
|------|------|
| 統計檢視 | 圓餅圖＋每日堆疊長條圖（預設） |
| MTBF/MTTR 分析圖 | 平均故障間隔／平均修復時間分析 |

### 統計檢視 · 整體圓餅圖

標題顯示所選 AGV 與日期區間（如 `AGV_001 2026-04-01 ~ 2026-06-15`）。

| 狀態 | 色標 | 說明 |
|------|------|------|
| RUN | 綠 | 執行中 |
| IDLE | 黃 | 閒置 |
| DOWN | 紅 | 當機／異常 |
| CHARGE | 藍 | 充電 |
| UNKNOWN | 灰 | 未知或未分類 |

右上方 **單位** 下拉（如 **小時**）控制圖表時間單位。圓餅各扇區標示該狀態累計時數（如 `IDLE: 13 hour`）。

### 統計檢視 · 每日長條圖

**統計資料** 區塊以堆疊長條圖顯示各日 RUN、IDLE、DOWN、CHARGE 時間：

- **X 軸**：日期（如 2026/04/14、2026/05/13）
- **Y 軸**：時間數值（依單位與資料尺度）
- 可對照圓餅圖，找出特定日期 IDLE 或 DOWN 異常偏高的日子

## 操作步驟

### 查詢單車稼動率

1. 選擇 **AGV 名稱**
2. 設定 **起始日期** 與 **結束日期**
3. 點選 **查詢**
4. 於 **統計檢視** 檢視圓餅圖與每日長條圖

### 切換時間單位

1. 在圓餅圖區域選擇 **單位**（如小時）
2. 確認扇區標籤與長條圖刻度已更新

### 檢視 MTBF／MTTR

1. 點選 **MTBF/MTTR 分析圖** 分頁
2. 分析該車在區間內的平均故障間隔與修復時間趨勢
3. 可搭配 [警報歷史查詢](/docs/guide/data-query/alarm-history-query) 對照 DOWN 時段

### 交叉分析

1. 若某日出現 DOWN 時間偏高，記下日期
2. 至 [警報歷史查詢](/docs/guide/data-query/alarm-history-query) 篩選同日警報
3. 至 [任務歷史查詢](/docs/guide/data-query/task-history-query) 確認是否因任務失敗導致停機

## 注意事項

- 統計結果依 AGVS 紀錄的狀態切換計算，與 [車輛狀態](/docs/guide/basics/vehicle-status) 即時顯示可能因時間粒度不同而略有差異
- 日期區間過長可能使長條圖較密集，建議依報表需求分段查詢
- **UNKNOWN** 佔比過高時，請聯絡工程人員確認狀態紀錄是否正常
- IDLE 佔比高不一定代表異常，需結合派工模式（[連線與模式切換](/docs/guide/basics/modes-change)）與產線負載一併評估
- MTBF／MTTR 供趨勢參考，正式 KPI 請依現場定義與 SOP 解讀
