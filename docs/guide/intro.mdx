# 網頁操作概覽

本區說明 **GPM AGVS** 派車系統 Web 操作介面的使用方式。前端為 **GPM AGVS** 網頁應用（`AGVSystem` 專案 `wwwroot`），供現場人員監控派車、設備與系統設定。

## 適用對象

| 角色 | 建議閱讀 |
|------|----------|
| 現場操作員 | 基本操作、派車監控、設備監控 |
| 工程／維護人員 | 告警處理、系統設定 |
| 系統管理員 | 使用者管理、SECS/GEM 設定、軟體更新 |

## 文件結構

| 章節 | 內容 |
|------|------|
| [基本操作](/docs/guide/basics/login) | 登入、首頁導覽 |
| [派車監控](/docs/guide/dispatch/main-dashboard) | 主畫面、任務、地圖與交管 |
| [設備管理](/docs/guide/equipment/equipment-monitoring) | 設備狀態、料架、充電站 |
| [告警處理](/docs/guide/maintenance/alarm-handling) | 告警查詢與處置 |
| [系統設定](/docs/guide/settings/user-manager) | 使用者、SECS/GEM、軟體更新 |

## 存取方式

```
http://{AGVS_HOST}/
```

實際網址與 Port 依現場部署。首次使用請向管理員取得帳號與權限。
