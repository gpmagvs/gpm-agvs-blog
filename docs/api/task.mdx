# 任務 API

## 列出任務

`GET /tasks`

查詢派車任務列表，支援分頁與狀態篩選。

### Query 參數

| 參數 | 類型 | 說明 |
|------|------|------|
| `status` | string | 篩選狀態：`pending`、`assigned`、`in_progress`、`completed`、`cancelled`、`failed` |
| `from` | string | 起始時間（ISO 8601） |
| `to` | string | 結束時間（ISO 8601） |
| `page` | number | 頁碼，預設 1 |
| `pageSize` | number | 每頁筆數，預設 20 |

### Request 範例

```bash
curl -X GET "https://api-staging.example.com/v1/tasks?status=completed&page=1" \
  -H "Authorization: Bearer <token>"
```

### Response 範例

```json
{
  "data": [
    {
      "taskId": "TASK-20250615-001",
      "status": "completed",
      "origin": "ST-A01",
      "destination": "ST-B03",
      "assignedVehicle": "AGV-003",
      "createdAt": "2025-06-15T10:30:00Z",
      "completedAt": "2025-06-15T10:45:00Z"
    }
  ],
  "meta": {
    "page": 1,
    "pageSize": 20,
    "total": 1
  }
}
```

## 任務狀態說明

| 狀態 | 說明 |
|------|------|
| `pending` | 已建立，等待分配車輛 |
| `assigned` | 已分配車輛，等待開始 |
| `in_progress` | 車輛執行中 |
| `completed` | 任務完成 |
| `cancelled` | 已取消 |
| `failed` | 執行失敗 |

:::info Webhook（規劃中）
未來版本將支援任務狀態變更 Webhook 通知，無需輪詢 API。
:::
