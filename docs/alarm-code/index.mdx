# Alarm Code

本節彙整 **派車系統（AGVS／VMS）** 與 **車載系統（AGV／VCS）** 的異常碼（Alarm Code）說明，供現場排查、警報歷史查詢與 [即時警報](/docs/guide/realtime-alarm/overview) 對照使用。

## 章節導覽

| 系統 | 說明 | 連結 |
|------|------|------|
| 派車系統 Alarm Code | AGVS／VMS 端定義的系統、設備與任務相關異常碼 | [派車系統](/docs/alarm-code/dispatch-alarm-code) |
| 車載系統 Alarm Code | AGV 車載端定義的硬體、導航、交握等異常碼 | [車載系統](/docs/alarm-code/vehicle-alarm-code) |
