# 頂部狀態列


## 功能概述

頂部狀態列固定於 [主畫面](/docs/vehicle-guide/main-screen) 最上方，由 **資訊列** 與 **電池狀態列** 兩層組成。資訊列以彩色區塊呈現車型、AGV 編號、使用者、版本與系統選單；電池列即時顯示電量百分比與電壓，供現場快速掌握車輛身分與電力狀態。

## 進入方式

進入主畫面或任何含頂部狀態列的車載頁面時，頂部狀態列 **固定顯示**，不隨中央分頁切換而消失。

## 畫面說明

![頂部狀態列](/img/vcs_webui/header-status.png)

### 資訊列（上排）

由左至右各區塊如下：

| 區塊 | 範例值 | 說明 |
|------|--------|------|
| 車型名稱 | GPM FORK AGV | 依 AGV 型別顯示（Fork、YTFORK、Inspection 等）；點選可切換 UI 版本顯示 |
| AGV 編號 | AGV_002 | 目前車輛名稱；背景色隨 SubState 變化 |
| 使用者 | OPERATOR | 目前登入帳號；未登入時為預設訪客名稱，見 [使用者登入](/docs/vehicle-guide/user-login) |
| 版本 | F3.4.18.25 | 車載 APP 版本；雙擊可切換顯示 UI 版本 |
| 語系 | EN | 中文／英文切換；切換後介面文字更新 |
| SYSTEM | 下拉選單 | 全螢幕、關機等系統級操作 |

#### SYSTEM 選單

| 專案 | 許可權 | 說明 |
|------|------|------|
| 全螢幕切換 | 所有使用者 | 進入／退出瀏覽器全螢幕 |
| 車輛定位 | God | 開啟定位對話方塊 |
| Sick::LidarLoc Website | God | 開啟 Sick 定位工具網頁 |
| 車載更新 | God | 上傳車載更新檔 |
| 車載系統重啟 | God | 重啟車載系統 |
| 關機 | 所有使用者 | 關閉車載電腦（需確認） |

God 使用者左上角另可見 **設定** 圖示（⚙），開啟系統引數設定。

### 電池狀態列（下排）

| 專案 | 範例值 | 說明 |
|------|--------|------|
| 電池圖示 | 🔋 | 充電中時顯示充電圖示 |
| 電量 | 99% | 進度條與百分比；車載系統連線中斷時可能顯示異常動畫 |
| 電壓 | [0] 0.00 V | 電池編號與電壓；充電中另顯示充電電流 |

點選電池進度條可開啟 **電池資訊** 側邊抽屜，檢視詳細 BMS 資料。

### 特殊狀態顯示

| 狀態 | 外觀 | 說明 |
|------|------|------|
| 車載系統連線中斷 | 版本區塊變紅、車型／使用者區塊異常樣式 | 車載系統連線中斷 |
| 維護模式 | 中央顯示「維護模式」與關閉按鈕 | 取代一般資訊列布局 |
| SubState 異常 | AGV 編號區塊特殊底色 | 如 DOWN、ERROR 等 |

## 操作步驟

### 確認車輛與登入狀態

1. 確認 **AGV 編號** 是否為預期車輛
2. 檢視 **使用者** 欄位：顯示 `OPERATOR` 表示尚未登入
3. 需要進階操作時，至 [側邊操作區](/docs/vehicle-guide/side-panel) 登入

### 切換語系

1. 點選語系開關（**EN**／**中文**）
2. 介面切換為英文或繁體中文
3. 底部通知列 briefly 顯示語言變更結果

### 使用 SYSTEM 選單

1. 點選紅色 **SYSTEM** 按鈕
2. 選擇所需功能（如 **全螢幕切換**）
3. 關機、重啟等 destructive 操作需二次確認

### 檢視電池詳情

1. 點選電池進度條
2. 右側抽屜顯示各電池詳細狀態
3. 關閉抽屜返回主畫面

## 注意事項

- 頂部 **使用者** 僅顯示帳號名稱，登入／登出操作在 [側邊操作區](/docs/vehicle-guide/side-panel) 或 [使用者登入](/docs/vehicle-guide/user-login) 對話方塊
- 車載系統連線中斷時版本區塊變 **紅色**，並可能伴隨畫面底部連線錯誤提示
- 版本欄位 **雙擊** 為 God Mode 密技入口，非一般使用者請勿操作
- 不同 AGV 型別左側 **車型名稱** 文字不同（如 GPM Inspection AGV、GPM Parts AGV）
- Mini AGV 可能顯示 **雙電池** 進度條，佈局與 Fork 單電池略有差異
