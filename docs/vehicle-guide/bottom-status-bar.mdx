# 底部狀態列

對應前端模組：`QuicklyActions`、`CheckParamSettingButton`、`DiskStatus`

## 功能概述

底部狀態列固定於 [主畫面](/docs/vehicle-guide/main-screen) 最下方，顯示 **貨物 ID 讀取開關**、**系統參數確認**、**雷射模式**、**車速指令** 以及 **CPU／RAM／磁碟** 等即時資源資訊。此列不負責頁籤切換（頁籤位於中央區域上方），而是提供輔助設定與系統健康監控。

:::info Version2 介面
Version2 主畫面底部改為 `HomeviewFooter`，含模式開關、快捷圖示與 EMO 等；本節以 Classic 主畫面（`QuicklyActions`）為主，概念與資訊項目相同。
:::

## 進入方式

進入主畫面後，底部狀態列 **固定顯示** 於中央分頁內容下方（高度約 38px），切換狀態／操作／地圖等頁籤時保持可見。

若系統參數 `IsQuicklyActionFooterDisplay` 設為 false，此列可能隱藏。

## 畫面說明

![底部狀態列](/img/vcs_webui/footer-status.png)

由左至右各項目如下：

### 左側：貨物 ID 讀取

| 項目 | 範例值 | 說明 |
|------|--------|------|
| 貨物ID讀取 Cargo ID Read | OFF（紅） | 控制 CST Reader 是否自動觸發讀取貨物 ID |

| 狀態 | 說明 |
|------|------|
| ON（綠） | 啟用取貨時自動讀取貨物 ID |
| OFF（紅） | 關閉自動讀取 |

未登入時此開關是否顯示，依系統參數 `CstReaderSwitchDisplayWhenNotLogin` 決定；已登入使用者一律可見。

### 右側：系統資訊與設定

| 項目 | 範例值 | 說明 |
|------|--------|------|
| 確認參數設定 | 按鈕文字 | 開啟參數確認對話框，檢查取貨拍照、貨物偵測等開關是否符合 SOP |
| Laser | Bypass(0) | 目前雷射避障模式，與 [主畫面](/docs/vehicle-guide/main-screen) 狀態分頁同步 |
| Stop | （標籤） | 目前機器人速度指令（`CurrentRobotSpeedCommand`） |
| CPU | 63.4% | 車載電腦 CPU 使用率 |
| RAM | 273 Mb | 記憶體使用量 |
| 磁碟 | 已使用: 10.99 GB / 18.62 GB | 系統磁碟空間；滑鼠停留可見詳細 tooltip |

#### 確認參數設定對話框

點選 **確認參數設定** 後，系統載入並列出 SOP 相關參數，分為兩類：

| 分類 | 說明 | 範例項目 |
|------|------|----------|
| 無特殊需求 **[需開啟]** | 預設應為 ON 的安全／品質項目 | 取貨須拍照、貨物狀態偵測、取放貨前後檢查等 |
| 無特殊需求 **[需關閉]** | 預設應為 OFF 的項目 | 空取空放 |

不符合 SOP 的項目會以 **紅色標籤** 醒目提示。修改開關需 **Developer（Role ≥ 2）** 以上權限。

#### 磁碟狀態色標

| 使用率 | 圖示顏色 |
|--------|----------|
| ≤ 70% | 綠色（正常） |
| 70%～90% | 黃色（警告） |
| > 90% | 紅色（危急） |

## 操作步驟

### 切換貨物 ID 讀取

1. 確認已具備操作權限（已登入或系統允許訪客顯示）
2. 點選 **Cargo ID Read** 開關，切換 ON／OFF
3. 等待設定寫入後端；成功時顯示通知「讀取貨物ID功能已開啟／關閉」
4. 若寫入失敗，開關會自動還原

### 確認系統參數

1. 點選 **確認參數設定**
2. 等待系統參數載入完成
3. 檢查 **[需開啟]** 項目是否均為 ON、**[需關閉]** 項目是否均為 OFF
4. 若有紅色標籤項目，依 SOP 調整（需 Developer 以上權限）
5. 修改後系統自動儲存並顯示結果通知

### 監控系統資源

1. 定期留意 **CPU**、**RAM** 是否異常飆高
2. 查看 **磁碟** 使用量；超過 70% 建議清理 log 或暫存
3. 磁碟空間不足可能觸發 [即時警報顯示](/docs/vehicle-guide/realtime-alarm-display) 或 `SystemErrorNotify` 全螢幕警告

## 注意事項

- 底部 **Laser** 為唯讀顯示，修改雷射模式需依 SOP 於其他介面或參數設定
- **Stop** 標籤反映目前速度指令，供除錯參考，非操作按鈕
- 貨物 ID 讀取開關變更會寫入車載系統參數，影響取貨流程，請勿随意切換
- 後端參數載入失敗時，Reader 開關無法同步，會跳出「系統參數異常」警告
- 本列與 [側邊操作區](/docs/vehicle-guide/side-panel) 的 Online／Auto 模式 **獨立**；Version2 介面將模式開關移至底部 `HomeviewFooter`
- 磁碟使用率過高可能導致車載系統不穩定，請依維護 SOP 定期清理
