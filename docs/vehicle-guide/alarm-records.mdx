# 異常紀錄

對應前端模組：`AlarmWarningTable`、`AlarmTableAPI`

## 功能概述

**異常紀錄** 分頁提供車載端 Alarm／Warning **歷史列表** 的查詢、篩選與分頁瀏覽。與 [即時警報顯示](/docs/vehicle-guide/realtime-alarm-display) 不同，本頁可回溯過去數月紀錄，並支援依類型、時間區間、異常碼篩選；登入使用者可清除紀錄或編輯 Alarm Table。

:::tip 派車系統警報歷史
若需查詢 **全場 AGV、充電站、VMS** 等跨設備警報歷史（含任務名稱、發生地點、CSV 匯出），請使用派車系統 [警報歷史查詢](/docs/guide/data-query/alarm-history-query)。
:::

## 進入方式

1. 進入 [主畫面](/docs/vehicle-guide/main-screen)
2. 點選中央頁籤 **異常紀錄**
3. 切換至該頁籤時系統自動重新載入資料

路由 `/alarm` 亦指向同一元件。

## 畫面說明

![異常紀錄](/img/vcs_webui/alarms-history-query.png)

### 工具列

| 項目 | 說明 |
|------|------|
| ⚙（滑桿圖示） | 開啟右側 **篩選設定** 抽屜 |
| 異常類型 | 下拉選單：ALL／Alarm／Warning |
| 刷新 | 重新向後端查詢目前條件下的資料 |
| 清除異常紀錄 | 登入後可見；清除車載端全部 Alarm 紀錄（需確認） |
| 編輯 Alarm Table | 登入後可見；前往 Alarm Table 編輯頁 |

### 分頁

表格上方顯示 **prev / pager / next** 分頁控制，每頁 **15** 筆，依總筆數翻頁。

### 結果表格

| 欄位 | 說明 |
|------|------|
| Time | 警報發生時間（格式 `yyyy/MM/DD HH:mm:ss`） |
| Code | 異常碼 |
| Description | 警報描述；中文介面顯示 CN，英文介面顯示 Description |
| Level | 等級：`Alarm` 或 `Warning` |

#### 列背景色標

| Level | 背景色 |
|-------|--------|
| Alarm | 淺紅色 |
| Warning | 淺黃色 |
| 其他 | 淺綠色 |

### 篩選設定抽屜

點選 ⚙ 開啟右側抽屜，可設定：

| 項目 | 說明 |
|------|------|
| 顯示警報類型 | ALL／Alarm／Warning |
| 時間區間設定 | **不限時間**（預設近三個月）或 **指定時間**（日期時間區間選擇器） |
| 異常碼 | 依 Alarm Table 分類篩選特定 Code |

快捷時間：前一周、前一個月、前三個月。

## 操作步驟

### 查詢異常紀錄

1. 進入 **異常紀錄** 頁籤
2. 選擇 **異常類型**（如 Alarm）
3. 點選 **刷新**
4. 使用分頁瀏覽結果

### 依時間或異常碼篩選

1. 點選 ⚙ 開啟篩選抽屜
2. 選 **指定時間** 並設定起迄，或選 **異常碼**
3. 變更後自動重新查詢
4. 關閉抽屜檢視表格

### 清除異常紀錄

1. 確認已 [登入](/docs/vehicle-guide/user-login)
2. 點選 **清除異常紀錄**
3. 在確認對話框中選 **OK**
4. 紀錄清除後表格重新載入

:::warning
清除異常紀錄 **不等同** 異常復歸；請先排除根因，再清除紀錄。
:::

### 對照即時警報

1. 右下方 [即時警報顯示](/docs/vehicle-guide/realtime-alarm-display) 記下 Alarm Code
2. 至本頁以 **異常碼** 篩選或翻頁查找
3. 確認發生時間與 Level

## 注意事項

- 預設 **不限時間** 實際查詢近 **三個月** 資料；更早紀錄請指定時間區間
- **清除異常紀錄** 僅登入使用者可見；訪客（OPERATOR）無法清除
- 本頁為 **車載本地** Alarm 紀錄；派車系統彙整的跨設備歷史見 [警報歷史查詢](/docs/guide/data-query/alarm-history-query)
- 清除紀錄不會自動解除 [即時警報顯示](/docs/vehicle-guide/realtime-alarm-display) 中仍作用中的警報
- 異常復歸請使用 [側邊操作區](/docs/vehicle-guide/side-panel) 的 **異常排除** 按鈕
- **編輯 Alarm Table** 為 God／維護功能，修改前請依變更流程操作
