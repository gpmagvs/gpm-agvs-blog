# 地圖顯示

對應前端模組：`TaskDelivery`、`MapShow`（OpenLayers）、`AGVLocalization`

## 功能概述

**地圖顯示** 分頁以 OpenLayers 呈現現場 **路徑、站點（Tag）與 AGV 即時位置**，並支援從派車系統同步圖資、縮放平移，以及點選站點派發 **本地任務**（移動、取貨、放貨、充電等）。同一地圖元件亦嵌入 [操作 → 移動控制](/docs/vehicle-guide/operations) 下方，功能一致。

## 進入方式

1. 進入 [主畫面](/docs/vehicle-guide/main-screen)
2. 點選中央頁籤 **地圖顯示**
3. 地圖載入完成後即可檢視路網與 AGV 位置

## 畫面說明

![地圖顯示](/img/vcs_webui/map-view.png)

### 地圖工具列

| 控制項 | 圖示／文字 | 說明 |
|--------|------------|------|
| 重置視角 | 🏠（house） | 恢復地圖預設縮放與位置（`OriginalZomm`） |
| 定位至 AGV | 📍（geo-alt） | 將視角移至目前 AGV 位置（`GoToAGVLoc`） |
| 重新下載圖資 | 從派車系統重新下載圖資 | 向 AGVS 重新同步地圖資料；執行中顯示 loading |
| MAP | Map_UMTC_3F ✓ | 目前使用中的地圖名稱；輸入框為唯讀狀態 |

### 地圖縮放

地圖左側浮動 **＋**／**－** 按鈕，用於放大／縮小檢視範圍。

### 地圖圖層（截圖範例）

| 元素 | 說明 |
|------|------|
| 灰色路徑 | 站點間可行駛路線 |
| 黃色圓點 | 站點（Tag 節點），標示編號如 7、9、23、27、33、65、67 |
| 綠色文字 | 站點／區域名稱，如 PRC#1～#5、Line#A/B/C Load／Unload |
| RACK 標籤 | 料架位置，如 RACK02L_1,5,9 |
| 維修站點 | 如 **維修站點63**（節點 63 附近） |
| 三角形圖示 | Load／Unload 等功能站方向指示 |
| AGV 圖示 | 目前車輛位置與狀態（若顯示） |

左側 [側邊操作區](/docs/vehicle-guide/side-panel) **Tag**（截圖為 **23**）可與地圖節點對照，確認 AGV 所在或讀取 Tag。

### 點選站點：本地任務抽屜

於 **地圖顯示** 分頁點選地圖站點後，底部彈出 **Task Delivery** 抽屜。抽屜標題顯示 **站點名稱 (Tag-n)** 與 **座標** `[x, y]`；右上方 **X** 可關閉。

依站點圖資屬性，抽屜內容分為三種類型：

| 站點類型 | 層數選擇 | 可用按鈕 |
|----------|----------|----------|
| 一般站點 | 無 | 移動 |
| 設備站點（LD/ULD） | 有（第一～三層） | 移動、放貨、取貨 |
| 充電站／含充電之設備站 | 有（若為 LD/ULD 站） | 移動、放貨、取貨、充電（或交換電池） |

點選按鈕後跳出 **Task Delivery** 確認對話框；若選取 LD/ULD 且指定層數，確認文字會包含 **層數** 資訊。

#### 一般站點

![一般站點抽屜](/img/vcs_webui/normal-station-only-move.png)

僅具 **通行／停靠** 功能、不支援取放貨或充電的站點。抽屜僅 **移動** 按鈕可用（青色），其餘按鈕不顯示或為禁用狀態。

| 項目 | 範例 |
|------|------|
| 站點 | `1 (Tag-1)` |
| 座標 | `[2.876, 0]` |
| 操作 | **移動** — 命令 AGV 導航至此 Tag |

#### 設備站點（取放貨）

![設備站點抽屜](/img/vcs_webui/eq-station-load-unload.png)

支援 **Load／Unload（放貨／取貨）** 的設備站或 Port，抽屜上方顯示 **層數** 選擇。

| 項目 | 範例 |
|------|------|
| 站點 | `Line#C Unload (Tag-44)` |
| 座標 | `[-1.36, -1.142]` |
| 層數 | 第一層／第二層／第三層（必選其一，預設第一層） |

| 按鈕 | 顏色 | 說明 |
|------|------|------|
| 移動 | 青色 | 導航至該站 |
| 放貨 | 橘色 | Load — 放貨至設備 |
| 取貨 | 深藍 | Unload — 自設備取貨 |

:::info
設備站若 **不** 含充電功能，**充電** 按鈕不會啟用。
:::

#### 充電站／含充電之設備站

![含充電站點抽屜](/img/vcs_webui/contain-charge-station.png)

充電站，或同時支援 **取放貨 + 充電** 的設備站（如部分 RACK 站）。除移動與取放貨外，另顯示 **充電** 按鈕；若為換電站則顯示 **交換電池** 取代充電。

| 項目 | 範例 |
|------|------|
| 站點 | `RACK01R_2,4,6 (Tag-4)` |
| 座標 | `[2.057, -1.813]` |
| 層數 | 第一層／第二層／第三層（LD/ULD 站需選層） |

| 按鈕 | 顏色 | 說明 |
|------|------|------|
| 移動 | 青色 | 導航至該站 |
| 放貨 | 橘色 | Load |
| 取貨 | 深藍 | Unload |
| 充電 | 綠色 | 前往充電 |
| 交換電池 | 灰藍 | 換電任務（換電站專用，取代充電） |

Inspection AGV 於抽屜另顯示 **定位** 按鈕（非上列三種抽屜截圖所示）。

### 按鈕啟用邏輯（參考）

| 屬性 | 對應按鈕 |
|------|----------|
| `SelectedFeatureMovable` | 移動 |
| `SelectedFeatureLDULDable` | 放貨、取貨、層數 |
| `SelectedFeatureChargable` | 充電 |
| `SelectedFeatureBatExchangable` | 交換電池（優先於充電） |
| `SelectedFeatureIsVirtualPt` | 虛擬點不可為終點，**移動** 禁用 |

## 操作步驟

### 檢視 AGV 與路網

1. 進入 **地圖顯示** 頁籤
2. 確認 **MAP** 名稱是否為預期樓層／區域（如 `Map_UMTC_3F`）
3. 點 📍 **定位至 AGV** 將視角移至車輛
4. 使用 **＋／－** 調整縮放，檢視站點編號與 Line／RACK 標籤

### 重新下載圖資

1. 確認 [側邊操作區](/docs/vehicle-guide/side-panel) **派車系統** 連線正常（建議綠燈）
2. 點選 **從派車系統重新下載圖資**
3. 等待 loading 完成；地圖重新載入
4. 若失敗，檢查 AGVS 連線與地圖名稱設定

### 派發本地任務

1. 確認 **Offline** + **Manual** 且已 [登入](/docs/vehicle-guide/user-login)（依任務類型與 SOP）
2. 點選地圖上目標 **站點**，底部抽屜依站點類型顯示（見上方三種範例）
3. **設備站／含充電站**：先選 **層數**（第一～三層）
4. 點選 **移動**／**取貨**／**放貨**／**充電** 等可用按鈕
5. 在 **Task Delivery** 確認對話框中確認 Action、Tag 與層數
6. 任務派送後可至 **狀態** 分頁或側邊 SubState 追蹤

### 重置視角

地圖拖移或縮放过远时，点 🏠 **重置視角** 恢复默认视图。

## 注意事項

- **從派車系統重新下載圖資** 依赖 AGVS 连线；派車系統断线时可能失败
- 本地任務與派車系統派工不同；Online + Auto 模式下請勿随意派送本地任務
- **虛擬點不可為終點站**；点选虚拟点时会提示
- 地圖編輯（新增／移除站點、點位設定）為 **edit_mode** 功能，一般現場操作頁 `task_allocatable=true` 時以派任務為主
- [移動控制](/docs/vehicle-guide/operations) 頁下方嵌入較小地圖，工具列與操作相同
- 定位異常時，地圖上 AGV 位置可能與實際不符，請先處理 [狀態](/docs/vehicle-guide/main-screen) 分頁的定位問題
- Inspection AGV 可透過抽屜 **定位** 或 `AGVLocalization` 進行定位輔助
