# 操作

對應前端模組：`AgvOperator`、`AgvcControl`、`ZAxisControl`、`IOTable`

## 功能概述

**操作** 分頁提供 AGV **手動控制** 功能，含移動、牙叉（Z 軸／水平）、Digital Input／Output 監控與切換。所有手動操作預設需 **已登入**、**Offline**、**Manual** 模式；God 角色不受模式限制。

## 進入方式

1. 進入 [主畫面](/docs/vehicle-guide/main-screen)
2. 點選中央頁籤 **操作**
3. 確認 [側邊操作區](/docs/vehicle-guide/side-panel) 為 **Offline** + **Manual**，並完成 [用戶登入](/docs/vehicle-guide/user-login)

操作分頁內再以子頁籤切換：**移動控制**、**牙叉控制**（Fork 車型）、**Input**、**Output**。

## 共通前提

| 條件 | 說明 |
|------|------|
| 已登入 | 未登入時控制區顯示灰色遮罩與提示文字 |
| Offline | Online 模式下無法手動控車 |
| Manual | Auto 模式下無法手動控車 |
| God 角色 | Role 3 不受 Online／Auto 限制 |

切換離開 **移動控制** 子頁籤時，系統會自動 **force_stop** 停止鍵盤移動。

## 移動控制

對應子頁籤：**移動控制**（`AgvcControl`、`AgvcControlPanel`、`TaskDelivery`）

### 功能概述

手動驅動 AGV 前進、後退、轉向與斜向移動，並顯示 Tag、偏移量、角度等定位資訊。下方嵌入 **地圖** 區塊，可檢視 AGV 位置並重新下載圖資。

### 畫面說明

![移動控制](/img/vcs_webui/move-control.png)

#### 左側：定位資訊

| 欄位 | 範例值 | 說明 |
|------|--------|------|
| Tag ID | 100 ✓ | 目標或目前讀取 Tag；Inspection 車型顯示 Last Visited Tag |
| 偏移量 X | 0 | BCR X 偏移 |
| 偏移量 Y | 0 | BCR Y 偏移 |
| 角度 | 90 | BCR 角度 Theta |
| 位置上報 | Upload | 登入且 enabled 時，上報座標至地圖 |

#### 右側：方向控制面板

3×3 九宮格方向鍵：

| 按鈕 | 動作 |
|------|------|
| 前左／前／前右 | 前進左斜、直進、前進右斜 |
| 左／停止／右 | 左移、停止（中央八邊形圖示）、右移 |
| 後左／後／後右 | 後退左斜、後退、後退右斜（Mini AGV 後斜向可能隱藏） |

未符合操作條件時，面板覆蓋灰色遮罩，提示：

> 請先登入權限並 Offline&Manual 才可操作手動控車

截圖中使用者為 **OPERATOR**（未登入），方向鍵因此不可操作。

#### 下方：地圖區塊

| 項目 | 說明 |
|------|------|
| AGV 圖示 | 顯示車名與 SubState，如 `AGV_002 (IDLE)` |
| ERROR 標籤 | 定位或載物異常時可能顯示 |
| 縮放／Home／定位 | 地圖操作工具 |
| 從派車系統重新下載圖資 | 重新同步地圖資料 |
| MAP 下拉 | 選擇地圖，如 `MAP No_load` |

### 速度設定

點選控制面板底部灰色區域可展開：

| 參數 | 範圍 | 說明 |
|------|------|------|
| Linear Speed | 0.001～1 | 線速度 |
| Rotation Speed | 0.01～0.3 | 旋轉速度 |

### 操作步驟

1. 確認 [側邊操作區](/docs/vehicle-guide/side-panel) 為 **Offline** + **Manual**
2. 完成 [用戶登入](/docs/vehicle-guide/user-login)（遮罩消失後方可控車）
3. 進入 **操作** → **移動控制**
4. 必要時調整 Linear／Rotation Speed
5. 點選方向鍵移動；完成後點 **停止**
6. 離開本子頁籤前請確認已停止

### 注意事項

- 切換離開 **移動控制** 時系統自動 **force_stop**
- 移動中請注意周遭人員與設備
- 地圖區塊功能與 [地圖顯示](/docs/vehicle-guide/map-display) 頁部分重疊

## 牙叉控制

對應子頁籤：**牙叉控制**（`ZAxisControl`；僅 Fork 車型 `agv_type == 0`）

### 功能概述

控制垂直（Vertical）牙叉升降、尋原點、極限 Sensor 狀態與 Bypass 設定。水平（Horizon）軸若啟用，布局與 Vertical 類似。

### 畫面說明

![牙叉控制](/img/vcs_webui/fork-control.png)

#### 狀態列（Vertical）

| 項目 | 範例值 | 說明 |
|------|--------|------|
| Position | 0.00 | 目前牙叉位置（cm） |
| State | -1 | 驅動器狀態 |
| ECode | 0 | 錯誤碼 |
| 安全防護 | 開（藍） | 垂直動作安全防護開關 |
| 牙叉位置校點 | 按鈕 | 開啟 FORK TEACH 校點抽屜 |
| 尋原點動作 | 按鈕 | 執行 Find Home |

#### 動作按鈕

| 按鈕 | 說明 |
|------|------|
| 上點位 | 移至上限預設點 |
| 上升 | 手動上升 |
| 原點 | 回 Home 位置 |
| 停止 | 立即停止（紅色停止圖示） |
| 下降 | 手動下降 |
| 下點位 | 移至下限預設點 |
| 上升搜尋 | 向上搜尋定位 |
| 下降搜尋 | 向下搜尋定位 |

#### Sensor 與 Bypass（截圖範例）

| 項目 | 範例值 | 說明 |
|------|--------|------|
| 極限 Bypass | No-Bypass（綠） | 硬體極限 Sensor Bypass 狀態 |
| 下壓 SensorBypass | No-Bypass（綠） | 下壓 Sensor Bypass 狀態 |
| 上極限 | ON（綠） | 上限 Sensor |
| Home | ON（綠） | 原點 Sensor |
| 下極限 | ON（綠） | 下限 Sensor |

左側 [側邊操作區](/docs/vehicle-guide/side-panel) 同步顯示 **牙叉高度**（如 0.000 cm）。

### 操作步驟

1. 確認 Offline + Manual 且已登入
2. 進入 **操作** → **牙叉控制**
3. 確認 **安全防護** 狀態符合 SOP
4. 點選動作按鈕；隨時可按 **停止**
5. 需校點時點 **牙叉位置校點** 開啟 FORK TEACH
6. 需尋原點時點 **尋原點動作**（AGV 運行中或 ONLINE 時禁用）

### 注意事項

- 未 enabled 時顯示：「請先登入權限並 Offline&Manual 才可操作 Z 軸與 bypass Hardware Limited 設定」
- **極限 Bypass**、**下壓 SensorBypass** 為危險操作，僅限受訓人員在 SOP 允許下使用
- ONLINE 狀態下無法執行 **尋原點動作**
- Inspection 等非 Fork 車型 **不顯示** 本子頁籤

## I/O-Input

對應子頁籤：**Input**（`IOTable`，`digital_type="input"`）

### 功能概述

即時顯示 WAGO 等 IO 模組 **Digital Input** 暫存器狀態，供除錯與狀態確認。Value 為 **1** 的列以藍底白字醒目顯示。

### 畫面說明

![I/O-Input](/img/vcs_webui/IO-Input.png)

#### 表格欄位

| 欄位 | 說明 |
|------|------|
| Address | DI 位址，如 `X0000`～`X000F` |
| Name | 信號名稱（部分位址可能空白） |
| Value | `0` 或 `1`；**1** 為藍底白字，**0** 為灰底 |

#### 截圖範例信號

| Address | Name | 說明 |
|---------|------|------|
| — | EMO | 急停輸入 |
| — | Bumper_Sensor | 前後膠條／Bumper |
| — | Panel_Reset_PB | 面板 Reset 按鈕 |
| — | Horizon_Motor_Switch | 水平馬達開關 |
| — | Vertical_Motor_Switch | 垂直馬達開關 |
| — | LeftProtection_Area_Sensor_3 | 左側保護區 Sensor |
| — | RightProtection_Area_Sensor_3 | 右側保護區 Sensor |

#### 分頁

表格左側為垂直分頁器（1、2、3、4…），每頁 **16** 筆 Address。

### 操作步驟

1. 進入 **操作** → **Input**
2. 使用左側分頁切換 Address 區段
3. 依 Name 查找目標信號
4. 依 Value **0／1** 確認感測器、按鈕等輸入狀態
5. 配合 [側邊操作區](/docs/vehicle-guide/side-panel) 確認 **IO 模組** 為綠燈（已連線）

### 注意事項

- Input 頁面以 **監控** 為主；手動切換 DO 請至 **Output** 子頁籤
- IO 模組斷線時數值可能不更新
- 登入 **ENG** 等工程帳號時頂部顯示對應使用者名稱，但 Input 頁本身不需 Toggle 操作

## I/O-Output

對應子頁籤：**Output**（`IOTable`，`isOutput="true"`）

### 功能概述

顯示 **Digital Output** 暫存器狀態，並在符合權限時以 **Toggle** 手動切換輸出。用於除錯、維護或 SOP 允許的 DO 測試。

### 畫面說明

![I/O-Output](/img/vcs_webui/IO-Output.png)

#### 表格欄位

| 欄位 | 說明 |
|------|------|
| Address | DO 位址，如 `Y0000`～`Y000F` |
| Toggle | 切換按鈕（需登入且該點位允許手動切換） |
| Name | 信號名稱 |
| Value | 目前 `0` 或 `1` |

#### 截圖範例信號

| Address | Name | 說明 |
|---------|------|------|
| Y0008 | Recharge_Circuit | 充電迴路 |
| Y0009 | Safety_Relays_Reset | 安全繼電器 Reset |
| Y000A | Horizon_Motor_Stop | 水平馬達 Stop |
| Y000B | Horizon_Motor_Free | 水平馬達 Free |
| Y000C | Horizon_Motor_Reset | 水平馬達 Reset |
| Y000D | Vertical_Motor_Stop | 垂直馬達 Stop |
| Y000E | Vertical_Motor_Free | 垂直馬達 Free |
| Y000F | Vertical_Motor_Reset | 垂直馬達 Reset |

#### 分頁

與 Input 相同，左側垂直分頁器，每頁 **16** 筆。

### 操作步驟

1. 確認已 [登入](/docs/vehicle-guide/user-login) 且 **Offline** + **Manual**（或 God 角色）
2. 進入 **操作** → **Output**
3. 使用左側分頁找到目標 Address
4. 點 **Toggle** 或 **雙擊列** 切換狀態
5. 若需確認，依 Warning 對話框操作

### 權限

| 角色 | 說明 |
|------|------|
| 已登入 + manualToggleEnable | 可 Toggle 允許的 DO |
| 未標記 manualToggleEnable | 僅 God 可強制 Toggle |
| manualToggleNeedConfirmed | 切換前跳出 Warning 確認 |

### 注意事項

- 錯誤 Toggle 可能導致馬達、繼電器等異常動作，請依 SOP 操作
- 不允許手動變更的 Output 會提示「This output is not allowed to be changed manually!」
- 截圖中使用者為 **ENG**，側邊 **登出** 為紅色，表示已登入工程帳號

## 其他子頁籤（進階）

| 子頁籤 | 顯示條件 | 說明 |
|--------|----------|------|
| 手動操作 | enabled | 雷射等手動參數設定 |
| Sensor/儀器控制 | Inspection AGV + enabled | 儀器車專用控制 |

## 共通注意事項

- 手動操作前務必確認 **Offline + Manual + 已登入**
- 任何子頁籤操作時請隨時注意 [即時警報顯示](/docs/vehicle-guide/realtime-alarm-display)
- 完成手動作業後，依 SOP 恢復 **Online + Auto** 以便派車
- EMO 急停位於 [側邊操作區](/docs/vehicle-guide/side-panel) 底部，緊急時優先使用
