# 側邊操作區


## 功能概述

側邊操作區位於 [主畫面](/docs/vehicle-guide/main-screen) 左側，是車載 HMI 的 **主要操作入口**。此區域集中放置初始化、異常處理、模式切換、連線監控、即時定位資訊與 **EMO 急停** 按鈕，現場人員無需切換頁籤即可完成多數日常操作。

## 進入方式

進入 [主畫面](/docs/vehicle-guide/main-screen) 後，側邊操作區 **固定顯示** 於畫面左側，所有中央分頁（狀態、操作、地圖等）共用同一側邊欄。

## 畫面說明

![側邊操作區](/img/vcs_webui/side-controls.png)

側邊操作區由上而下分為：**功能按鈕**、**模式切換**、**連線狀態**、**Tag／座標**、**牙叉高度／里程數** 與 **EMO** 六個區塊。

### 功能按鈕

| 按鈕 | 說明 |
|------|------|
| 初始化 | 啟動 AGV 初始化流程；執行中按鈕顯示「初始化中」，背景色隨 SubState 變化 |
| 異常排除 | 呼叫 Reset Alarm，清除車載端可復歸的異常；有警報時按鈕可能變紅 |
| 蜂鳴器關閉 | 關閉蜂鳴器；靜音時顯示 🔇 圖示，鳴叫中顯示 🔊 圖示 |
| 移除卡匣 | 清除車上卡匣／載物帳籍（Fork 等車型）；旁側徽章可顯示載物狀態 |
| 登入 | 開啟 Login 對話方塊；已登入時改為紅色 **登出**，詳見 [使用者登入](/docs/vehicle-guide/user-login) |

:::info Inspection AGV
Inspection 車型（Inspection 車型）不顯示 **移除卡匣** 按鈕。
:::

### 模式切換（截圖範例）

| 專案 | 範例值 | 說明 |
|------|--------|------|
| Online Mode | Offline（紅） | **Online**（綠／藍）：連線派車系統，可接收任務；**Offline**（紅）：離線，僅本地操作 |
| Auto Mode | Manual（紅） | **Auto**（綠／藍）：自動執行任務；**Manual**（紅）：手動模式，可進行 [操作](/docs/vehicle-guide/operations) 分頁控制 |

切換前系統會跳出確認對話方塊。非 IDLE 狀態下可能無法切換為 Online 或 Auto。

### 連線狀態（截圖範例）

標題為 **連線狀態**，以紅／綠圓點表示各子系統連線：

| 專案 | 範例 | 說明 |
|------|------|------|
| 派車系統 | ● 紅 | 與 AGVS 派車系統（VMS 連線）通訊狀態 |
| 車控系統 | ● 綠 | 車控系統連線狀態 |
| IO 模組 | ● 綠 | WAGO 等 IO 模組通訊狀態 |

:::note
連線狀態區塊在 IO 已連線且剎車釋放等條件下可能自動隱藏；異常時才顯示以便排查。
:::

### Tag 與座標（截圖範例）

| 專案 | 範例值 | 說明 |
|------|--------|------|
| Tag | 100 ✓ | BCR 讀取到的 Tag ID；0 或讀取失敗時可能顯示紅色 |
| 座標 | 13.09, -2.41 ⚠ | 車輛目前 X、Y 座標；定位異常（LocStatus ≠ OK）時顯示警告 |

IO 連線正常且剎車釋放時，Tag 可能改以 **大標籤** 顯示，並附加 **偏移量**（X／Y）欄位。

### 牙叉高度與里程數（截圖範例）

| 專案 | 範例值 | 說明 |
|------|--------|------|
| 牙叉高度 | 0.000 cm | Fork 車型專用；雙擊標題可進入 God Mode 相關設定 |
| 里程數 | 6.900 km | 車輛累計行駛里程 |

### EMO 急停

側邊欄最下方為大型紅色 **EMO** 按鈕，觸發軟體急停（Emergency Off）。車載系統連線中斷時可能被停用。

## 操作步驟

### 初始化 AGV

1. 確認車輛處於安全狀態（無人員在運動路徑上）
2. 點選 **初始化**
3. 在 **AGV Initialize** 確認對話方塊中點選 **OK**
4. 等待 SubState 由 `Initializing` 變為正常狀態（如 IDLE）
5. 若初始化失敗，依對話方塊訊息處理（如建立貨物帳籍）

初始化進行中再次點選，可取消初始化流程。

### 異常排除

1. 確認已知曉 [即時警報顯示](/docs/vehicle-guide/realtime-alarm-display) 或 [異常紀錄](/docs/vehicle-guide/alarm-records) 中的異常內容
2. 依 SOP 排除根因
3. 點選 **異常排除**（Reset Alarm）
4. 確認警報與 SubState 是否恢復正常

### 關閉蜂鳴器

1. 現場已知曉告警原因或正在處理中
2. 點選 **蜂鳴器關閉**
3. 蜂鳴器停止後，警報紀錄仍可能保留

### 切換 Online／Auto 模式

1. 確認 AGV 處於 **IDLE**（上線時充電中 CHARGING 亦可）
2. 點選 **Online Mode** 或 **Auto Mode** 開關
3. 在確認對話方塊中點選 **OK**
4. 等待模式切換完成；上線時可能顯示 **AGV Online Requesting** 等待對話方塊

手動操作前，建議設為 **Offline** + **Manual**，並先 [登入](/docs/vehicle-guide/user-login)。

### 移除卡匣

1. 確認車上已無實物貨物
2. 點選 **移除卡匣**
3. 在確認對話方塊中點選 **OK**
4. 依結果訊息確認是否成功；若車上仍有貨物會回傳錯誤

### 觸發 EMO

1. 僅在緊急情況下使用
2. 點選底部 **EMO** 按鈕
3. 依 SOP 復歸後再重新初始化

## 注意事項

- 車載系統連線中斷時，多數按鈕與模式開關會被 **停用**
- 系統初始化進行中時，**初始化** 與模式開關可能被停用
- **異常排除** 僅復歸車載端 Alarm，不等同於清除 [異常紀錄](/docs/vehicle-guide/alarm-records) 分頁中的歷史資料
- **蜂鳴器關閉** 僅停止聲音，不解除異常本身
- 派車系統連線為紅燈（Offline 且 AGVS 斷線）時，無法接收派車任務，但本地手動操作仍可能可用
- 座標旁出現警告圖示表示定位異常，請勿進行自動導航，依 SOP 重新定位
- Version2 介面部分快捷操作改至 [底部狀態列](/docs/vehicle-guide/bottom-status-bar)，但模式概念相同
- God 使用者另有進階維護選單，不在本側邊欄範圍內
